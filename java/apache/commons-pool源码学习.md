## Apache Commons Pool介绍

Apache Commons Pool有两个版本，1.x 和 2.x，2.x是对1.x的完全重写版本，一般项目都采用2.x（jdk版本1.6+），此工具的作用是做对象池，防止对象的重复创建，一般用在数据库连接池等使用场景。更具体的信息可以参考 [apache官方网站](http://commons.apache.org/proper/commons-pool/)
